首先将MainActivity 改成 继承自FragmentActivity(最好这样做)

public class MainActivity extends FragmentActivity

编写menu菜单栏,新建menu文件夹,新建menu.xml文件,对应底部导航的按钮

编写主界面，底部导航栏采用官方BottomNavigationView的Ex版本

创建与menu对应数量的XML布局文件（非new Activity）

创建与新建布局文件对应数量的class 继承自Fragment

继承自Fargment 重写onCreateView方法 其中View对象作为初始化控件的方法

创建自定义FragmentPagerAdapter

只需要写构造方法，并重写getItem() 和 getCount()方法即可

在MainActivity中构造自定义FragmentPagerAdapter，绑定创建的Fragment

最后对BottomNavigationViewEx做最后的设置，使其符合实际场景的需求
