Maven

Problem

最近遇到了一些问题

pom.xml引入他人项目时出现了错误，Could not find artifact xxx:jar:x.x.x
几种原因

  • 网络上没有该文件，本地仓库也没有（可能性较小）
  • 文件没有下全，网络有问题，删掉对应的包，reimport
  • 对应库文建夹下垃圾文件有重名jar包，只留一个jar文件即可
